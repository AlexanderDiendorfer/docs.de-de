# <a name="compiler-error-cs1626"></a><span data-ttu-id="efa07-102">Compilerfehler CS1626</span><span class="sxs-lookup"><span data-stu-id="efa07-102">Compiler Error CS1626</span></span>
<span data-ttu-id="efa07-103">Mit "yield" kann im Text eines try-Blocks mit einer catch-Klausel kein Wert zurückgegeben werden.</span><span class="sxs-lookup"><span data-stu-id="efa07-103">Cannot yield a value in the body of a try block with a catch clause</span></span>  
  
 <span data-ttu-id="efa07-104">Eine yield-Anweisung ist in einem try-Block nicht zulässig, wenn eine dem try-Block zugeordnete catch-Klausel vorhanden ist.</span><span class="sxs-lookup"><span data-stu-id="efa07-104">A yield statement is not allowed in a try block if there is a catch clause associated with the try block.</span></span> <span data-ttu-id="efa07-105">Um diesen Fehler zu vermeiden, verschieben Sie die Yield-Anweisung aus dem Try/Catch/finally-Block, oder entfernen Sie den CatchBlock.</span><span class="sxs-lookup"><span data-stu-id="efa07-105">To avoid this error, either move the yield statement out of the try/catch/finally block, or remove the catch block.</span></span>
  
 <span data-ttu-id="efa07-106">Im folgenden Beispiel wird CS1626 generiert:</span><span class="sxs-lookup"><span data-stu-id="efa07-106">The following sample generates CS1626:</span></span>  
  
```csharp  
// CS1626.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
         yield return this;  // CS1626  
      }  
      catch  
      {  
  
      }  
      finally
      {
      
      }
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
