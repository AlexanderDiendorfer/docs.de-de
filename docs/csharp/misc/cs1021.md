<span data-ttu-id="168d4-103">Die integrale Konstante ist zu groß.</span><span class="sxs-lookup"><span data-stu-id="168d4-103">Integral constant is too large</span></span>

    {
        int a = 18_446_744_073_709_552_000;
    }
}  
```

    {

    }
}
```
