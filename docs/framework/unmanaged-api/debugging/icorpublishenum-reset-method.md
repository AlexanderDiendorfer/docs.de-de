# <a name="icorpublishenumreset-method"></a>ICorPublishEnum::Reset-Methode
Verschiebt den Cursor auf den Anfang der Enumeration.  
  
## <a name="syntax"></a>Syntax  
  
```  
HRESULT Reset ();  
```  
  
## <a name="parameters"></a>Parameter  
 `Reset` nimmt keine Parameter.
