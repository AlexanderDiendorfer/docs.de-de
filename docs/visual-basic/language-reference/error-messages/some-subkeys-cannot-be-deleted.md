# <a name="some-subkeys-cannot-be-deleted"></a>Einige Unterschlüssel können nicht gelöscht werden
Es wurde versucht wurde, um einen Registrierungsschlüssel zu löschen, aber der Vorgang ist fehlgeschlagen, da einige Unterschlüssel können nicht gelöscht werden. In der Regel ist dies aufgrund fehlender Berechtigungen.  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Stellen Sie sicher, dass Sie über ausreichende Berechtigungen zum Löschen der angegebenen Unterschlüssel haben.
