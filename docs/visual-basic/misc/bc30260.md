# <a name="name-is-already-declared-as-declaration-in-this-declarationspace"></a>'\<Name >' wurde bereits deklariert als\<Deklaration >' in diesem \<Deklarationsabschnitt >
Im selben Deklarationsabschnitt (Klasse, Modul, Schnittstelle oder Struktur) sind zwei Programmierelemente mit demselben Namen deklariert.  
  
 **Fehler-ID:** BC30260
