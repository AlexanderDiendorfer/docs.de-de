# <a name="method-methodname-is-already-declared-in-interface-interfacename"></a>Methode '\<Methodenname >' ist bereits in der Schnittstelle deklariert'\<Schnittstellenname >'
Sie haben eine Schnittstellenmethode ein zweites Mal deklariert, wodurch diese doppelt angelegt wurde.  
  
 **Fehler-ID:** BC30377  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie die doppelte Deklaration.
