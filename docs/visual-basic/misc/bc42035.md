# <a name="bad-guid-format"></a>Ungültiges GUID-Format.
Eine GUID wurde in einem ungültigen Format angegeben.  
  
 **Fehler-ID:** BC42035  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Überprüfen Sie das GUID-Format.
