# <a name="typename-is-an-unsupported-type"></a>'\<Typname >' wird nicht unterstützt
Der angegebene Typ kann nicht von Visual Studio dargestellt werden.  
  
 **Fehler-ID:** BC30649  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Verwenden Sie einen unterstützten Typ.  
  
-   Entwickeln Sie einen neuen Typ.
